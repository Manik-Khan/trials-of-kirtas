# Historical Forge validation bundles

These files record earlier delivery bundles. `MANIFEST.sha256`, `SHA256SUMS`, and
`SHA256SUMS.txt` already contained missing-path and hash-mismatch results against `main` at
baseline commit `5bf8d07`; they are preserved byte-for-byte and do not verify current main.

`MANIFEST.json` describes the July 16 current-handoff file set. Its paths are maintained after
repository moves, but it is not a complete repository manifest.
